# Python Quiz Application

A simple and interactive quiz application developed using HTML, CSS, and JavaScript.  
This project helps beginners test their basic Python programming knowledge through multiple-choice questions.

---

## 📌 Features

- Simple and clean user interface
- Multiple-choice quiz questions
- Responsive quiz layout
- Beginner-friendly project
- Easy to customize questions
- Attractive design with background styling

---

## 🖼️ Preview

The quiz application displays Python-related questions with multiple answer options.

Example:

**Who developed Python Programming Language?**

- Wick van Rossum
- Rasmus Lerdorf
- Guido van Rossum 
- Niene Stom
